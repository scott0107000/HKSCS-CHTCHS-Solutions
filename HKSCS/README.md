# 支援新舊香港外字

《香港增補字符集》（HKSCS）舊版編碼方案為在臺灣系統Big5編碼上增加香港字，新版編碼則接軌國際ISO 10646標準。字言字型除了開發出可同時支援以上兩種編碼的字型，更進一步能夠支援《政府通用字庫》（GCCS）字集，確保新老文件都能正常顯示香港字。
