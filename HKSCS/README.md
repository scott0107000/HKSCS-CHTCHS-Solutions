# 支援新舊香港外字

《香港增補字符集》（HKSCS）舊版編碼方案為在臺灣系統Big5編碼上增加香港字，新版編碼則接軌國際ISO 10646標準。字言字型除了開發出可同時支援以上兩種編碼的字型，更進一步能夠支援《政府通用字庫》（GCCS）字集，確保新老文件都能正常顯示香港字。

## 字型
- `Fancy-KSong-HKSCS-Trial.ttf`：字言庫宋體HKSCS試用版
- `Fancy-HKai-HKSCS-Trial.ttf`：字言港楷體HKSCS試用版，此字型以香港字形為主，並以全字庫正楷體補全Big5+HKSCS。
本字體使用之源頭字型《自由香港楷書》（此處自由指自由軟體，以「共享創意署名4.0 國際授權條款」釋出）有字形度量不一致，字面不同的情況，本字型在整合時亦修正了此情況。惟僅特別修改常用字形表內字，罕用字仍有風格及字面不同的問題。
