# 兩岸橋

字言字型「兩岸橋」中文字型包括以下４種規格字型：

## ①Big5簡（Big5S）

使用字言字型Big5S規格字型可一鍵將繁體中文文本轉換為簡體中文。沒有對應簡體字則以原繁體字顯示。

文件：`Fancy-Song-Big5S-Trial.ttf`

![Big5S](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEhAA2MJTVLYtaQASo93qIQGvBvKwqFhbZTzPHrbGrDWTmBVPS_h2OXYfNAsLribj0sgk7OQs-JcJXEMf9yCEYuvGKv03nf9219q9z-2CeC2hynsX_91Cr8RC7mTc5Cp6BB910G7b-9hJIO417rewnNiFg8-74chKqZ4KjEQQpLAky3eXDaDcj6Al2H_nOpk/s1500/Big5S.jpg)

## ②大陸繁體（GBF）

使用字言字型GBF規格字型可一鍵將簡體中文文本轉換為繁體中文，字型內同時收錄GB/T 12345標準的103個一簡對多繁的繁體字。

文件：`Fancy-KSong-GBF-Trial.ttf`

![GBF](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEhW9QdYJEfFdilU-8_QwzB9J2m3IyWnCiVmh_9CeiEnZsD0wWlCFECLR1cYDfKFtElxWWSpMAKjdo9OXf85gFjnEe4vVJ8aOiGGtnbh9geI1EfiR7Fq47cnxlWEf54uQDkHXCaBk3ynj5kCxGeNHyj0kbvY0tdA9dLIfosYPS7NXOsgCRHrQJwsEFc90Z0i/s1500/GBF.jpg)

## ③繁體系統讀簡體碼（S2B5）

使用字言字型S2B5規格字型，可在繁體系統環境下正確顯示簡體GB的文本內容而不會亂碼，閱讀印刷皆宜。

文件：`Fancy-S2B5-Song-Trial.ttf`

![S2B5](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEhzZF1LcIvmhw7B13UL7ka9f77ZqTIGXFgatDcbkzvURWQIXsOrGNQ2dzB3xoAYFmTP05fVf26PUPJ9h0naZDcdtPq3zLNxoC-xNBewmjeo6feVBa7Moca8xDM5WN5jpTBomSFn-5QtMLQuVWU0BM3jKDCB3VYZYLMggu9cacq547EsSJJ7igJ3UI0mmLUf/s1500/S2B5.jpg)

## ④簡體系統讀繁體碼（T2GB）

使用字言字型T2GB規格字型，可在簡體系統環境下正確顯示繁體Big5的文本內容而不會亂碼，閱讀印刷皆宜。

文件：`Fancy-T2GB-Song-Trial.ttf`

![T2GB](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEiQatA7HaxRPvCqHGY0ud7MtTJBKcrLXrgGbcBiMWrJfKtHTO9Ks9iDE5YKicKnmlPj00Sf-ckjljy-g9Q_jF24ikaaPQ2L5ZidsLkrWs668sXbD_KyIcdxuguThgVSeY_IlTEqyMJJDtXJhLFEoImOSE0iCEC4yySv1dqzB0wQZ4Ef-7_9P0T_Tl7nVK1s/s1500/T2GB.jpg)

- `Fancy-Song-Big5S-Trial.ttf`：字言宋體Big5S試用版
- `Fancy-KSong-GBF-Trial.ttf`：字言库宋体繁-试用版
- `Fancy-S2B5-Song-Trial.ttf`：字言|Fancy-S2B5-Song-Trial`
- `Fancy-T2GB-Song-Trial.ttf`：字言|Fancy-T2GB-Song-Trial`
